### About

Vaultix builds payment infrastructure that enables businesses to move money across borders with speed, security and compliance. Our platform unifies fiat and digital asset rails into a single, developer-friendly ecosystem.

### What We Build

| Product | Description |
|---|---|
| **Vaultix Console** | Operations dashboard for payment management and monitoring |
| **Vaultix User** | End-user application for secure transactions and account management |
| **Vaultix Manager** | Affiliate and partner portal |
| **Vaultix SDK** | Node.js/TypeScript SDK for platform integration |
| **Vaultix MCP** | AI-ready connector for Claude and intelligent agents |

### Tech Stack

`TypeScript` `React` `Node.js` `Go` `PostgreSQL` `Ethereum` `Tron`

### Security

Security is foundational to everything we build. We follow industry best practices for key management, data encryption and access control. If you discover a vulnerability, please reach out to **security@vaultix.global**.
